1. HOW TO CREATE GMAIL ACT ?

  • Go to www.gmail.com.
  • Click on "Create account"
  • Fill in the details
  • Click "Next"
  • Verify your phone number
  • Enter the verification code
  • Provide recovery email, date of birth, and gender.
  • Agree to the terms
  • Finish setup

    • 2. HOW TO CREATE GITHUB ACT ?

  • Go to www.github.com.
  • Click on "Sign up"
  • Fill in the details
  • Verify your account
  • Choose your plan
  • Complete the setup

    • 3. HOW TO CONNECT VS CODE TO GITHUB ?

  • Install Git
  • Install VS Code
  • Launch Visual Studio Code.
  • Install GitHub extension: "GitHub Pull Requests and Issues"
  • Click on the Accounts icon on the Activity Bar on the left.
  • Click on "Sign in to GitHub" and follow the prompts to authenticate.

    • 4. HOW TO CLONE/PULL REPOSITORY FROM GITHUB ?

  • Go to the GitHub repository you want to clone and copy its URL.
  • Open VS Code:
  • Clone repository:
  • Open the Command Palette
  • Type "Git: Clone" and select it.
  • Paste the repository URL and choose a directory to clone the repository into.
  • Type "Git: Pull" and select it to pull updates from the repository.

    • 5. HOW TO PUSH REPOSITORY/FOLDER TO GITHUB ?

  • Navigate to your project folder in VS Code and initialize a Git repository
  • Stage your changes by running git add . in the terminal.
  • Commit your changes by running git commit -m "Initial commit".
  • Add the GitHub repository as a remote by running git remote add origin repository_url.
  • Push your changes to GitHub by running git push -u origin main.

    • 6. HOW TO PUBLISH A REPOSITORY AS LIVE SITE ?

  • Go to the repository on GitHub.
  • Click on "Settings".
  • Scroll down to the "GitHub Pages" section.
  • Select the branch you want to use as the source for your GitHub Pages site.
  • Choose the root directory for your site files.
  • Click "Save" to publish your site.
  • Your site will be available at https://username.github.io/repository.
